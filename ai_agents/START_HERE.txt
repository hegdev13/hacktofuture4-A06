TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════

❓ Dashboard won't load?
   → Verify port 3456 is available
   → Try different port: DASHBOARD_PORT=4000
   → Check firewall settings

❓ Connection failed?
   → Run: node test-live-metrics.js
   → Check ngrok status: https://refocus-cement-spud.ngrok-free.dev/status
   → Verify URL format (must be HTTPS)

❓ No pods detected?
   → Ensure ngrok endpoint returns JSON
   → Should return object with pods array

❓ Analysis not starting?
   → Click "Connect Metrics" button in dashboard
   → Check browser console for errors (F12)
